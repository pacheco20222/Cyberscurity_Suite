Welcome to the Cybersecurity Suite

This application demonstrates various data structures, their functionalities, and visual representations. Each structure has been integrated into a user-friendly interface for easy interaction and understanding. The idea is to provide hands-on experience with cybersecurity logs and data structures. For a company to train employees to identify and understand logs, this application can be used as a training tool.

Available Data Structures and Their Functionalities:

  • Arrays: Perform insertion, deletion, update, and access operations. Includes a regenerate script to reset array logs.
  • Stacks: Demonstrates LIFO (Last In, First Out) operations like push and pop. Comes with a regenerate script.
  • Queues: Demonstrates FIFO (First In, First Out) operations like enqueue and dequeue. Includes a regenerate script for resetting logs.
  • Binary Tree: Visualizes a binary tree structure with operations like insert, delete, and search. Features a regenerate script.
  • Graphs: Add nodes and edges, and compute the Minimum Spanning Tree (MST) using Kruskal's algorithm. Visualized graphically.
  • Doubly Linked List: Add, delete, and update nodes, with operations that emphasize the linked list's bidirectional structure. Includes a regenerate script.
  • Sorting: Demonstrates quicksort and binary search on a list of logs.
